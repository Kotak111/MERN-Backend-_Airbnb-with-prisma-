### Setup .env file

```bash


DATABASE_URL="your_mongo_url",
NEXTAUTH_SECRET="auth_secret"
GITHUB_ID="github_id",
GITHUB_SECRET="github secret"
GOOGLE_ID="google id"
GOOGLE_SECRET="google secret"
```

### Run this app locally

```shell
npm run build
```
